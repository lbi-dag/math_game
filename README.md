# Math Games

Small, no-ads math games built for quick practice sessions and classroom-friendly play.

## What's here
- Landing page: `index.html`
- About page: `about.html`
- Number Sense Sprint game: `number-sense-sprint.html` + `game.js`

## Run locally
Open `index.html` in your browser.

## Notes
- Main branch is hosted at `https://mathgames.win/` via CloudFlare.
- No build step required.
- Designed to be lightweight and easy to host on any static site.
